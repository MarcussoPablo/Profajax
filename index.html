O C# (leia-se C-Sharp), é uma linguagem de programação orientada a objetos,
que foi desenvolvida pela Microsoft e faz parte da plataforma .NET.
Embora a linguagem C# tenha sido criada do zero, foi baseada na linguagem
C++ e tem muitos elementos da linguagem Pascal e Java.
Java é uma linguagem de programação e plataforma computacional
lançada pela primeira vez pela Sun Microsystems em 1995.
Existem muitas aplicações e sites que não funcionarão,
a menos que você tenha o Java instalado, e mais desses
são criados todos os dias.
O Java é rápido, seguro e confiável.
De laptops a datacenters, consoles de games a
supercomputadores científicos, telefones celulares à Internet, o Java está em todos os lugares!
